# RED T7GER Website

Private local repo for the official RED T7GER artist website. Built with Vite, React, and TypeScript.

## Public Data Boundary

This site is intended for public release and should not include private personal data, credentials, internal campaign metrics, or unpublished strategy notes. Use only public artist identity, public streaming links, public social links, public press links, and approved artwork.

## Local Development

```bash
npm install
npm run dev
```

## Checks

```bash
npm run build
npm run lint
```

## Deployment Notes

The intended production domain is `redt7ger.com`. Update hosting DNS and the sitemap/robots files only when the site is ready to go public.
